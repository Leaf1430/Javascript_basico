O que é Lógica de Programação?

Lógica de programação é a capacidade de organizar o pensamento para criar instruções claras e sequenciais que um computador possa executar.

Analogia do sanduíche: Se você pedir “prepare o café” sem explicar os passos, a pessoa pode errar. Com o computador é igual: ele precisa de instruções detalhadas e na ordem correta.

Programar é traduzir problemas do mundo real em uma sequência lógica de comandos.

Pensamento Computacional

É uma abordagem para resolver problemas baseada em quatro pilares:

  • Decomposição: quebrar um problema grande em partes menores.
  • Reconhecimento de Padrões: identificar similaridades entre problemas.
  • Abstração: focar no essencial, ignorar detalhes irrelevantes.
  • Algoritmo: criar uma sequência passo a passo para a solução.

Exemplo: Organizar uma festa – decomponha em convites, comida, decoração; veja padrões de festas anteriores; abstraia o que é necessário; crie uma lista de tarefas.

Algoritmos no Dia a Dia

Um algoritmo é uma sequência finita de passos para realizar uma tarefa.

Exemplos:

  • Receita de bolo
  • Trajeto para a escola
  • Rotina matinal

Características: passos claros, ordem definida, finito (tem início e fim).

Fluxogramas

Representação visual de um algoritmo usando formas padronizadas:

  • Oval: início / fim
  • Retângulo: ação / processamento
  • Losango: decisão (sim/não)
  • Seta: fluxo / direção

Exemplo: Decidir o que vestir – se está frio, vista casaco; senão, vista camiseta.

Introdução ao JavaScript

JavaScript (JS) é a linguagem mais usada na web, roda no navegador.

Abra o console do navegador (F12) e digite:

console.log("Olá, mundo!");

Comentários:

// Comentário de uma linha /* Comentário de várias linhas */

Variáveis

Variável é um rótulo que guarda um valor na memória. Imagine uma caixa com etiqueta.

Usamos let e const para declarar.

let nome = "João"; const idade = 25;

Regras: nomes não podem começar com número, sem espaços, sem palavras reservadas. Use camelCase.

Atividade: Crie variáveis para seu nome, idade e cidade.

Tipos Primitivos

JavaScript tem três tipos primitivos principais:

  • String: texto (entre aspas)
  • Number: números (inteiros ou decimais)
  • Boolean: true ou false

Use typeof() para saber o tipo:

console.log(typeof("Olá")); // string console.log(typeof(42)); // number console.log(typeof(true)); // boolean

Mini-Diálogo no Console

Vamos criar um mini-diálogo usando variáveis e console.log:

let nome = "Maria"; let idade = 28; let cidade = "São Paulo"; let estudando = true; console.log("Olá, meu nome é " + nome + "!"); console.log("Tenho " + idade + " anos e moro em " + cidade + "."); console.log("Estudando JavaScript? " + estudando);
